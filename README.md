# ONLINE MUSIC STREAMING WEBSITE

## Technical Stack

### Back-end
* Spring Boot
* Spring Security
* Cloudinary
* OAuth 2.0 Authentication

### Front-end
* Thymeleaf
* HTML/CSS/JS
* Bootstrap
* jQuery

### Database
* MySQL

## User Types & Features

### Guest Users
* Basic music playback
* Search functionality
* Limited access to free content
* View artist profiles and albums

### Registered Users
* Personal account management
* Playlist creation and management
* Save favorite songs and albums
* Follow artists
* Basic streaming features

### Premium Users
* Ad-free listening experience
* Unlimited access to licensed music
* Exclusive content access

### Administrators
* Complete website management:
  * User account management
  * Content moderation
  * Music library management
  * Artist verification
  * Analytics and reporting
  * System configuration
  * Performance monitoring
  * Revenue tracking

### Business Partners
#### Artists/Labels
* Upload and manage music content
* Receive royalty reports
* Manage artist profiles
* Access analytics

#### Advertisers
* Campaign management
* Ad placement options
* Performance tracking

### Third-Party Partners
#### Payment Processors
* Secure payment processing
* Multiple payment methods support
* Transaction verification
* Payment status tracking
* Refund processing
* Subscription management
* Revenue sharing calculations
* Automated billing

## Premium Features
* Subscription management
* Billing history
* Payment processing
* Account upgrades/downgrades
